Hey there, This Project is used by mediapipe and openCv through python.
This mosutracking is a demo and easy way to implement using python language using normal laptop webcam.
Libraries used were:
1.opencv-pythonwebcam feed, drawing landmarks on frame
2.mediapipe - hand detection and landmark tracking
3.pyautogui - cursor movement and clicking
4.pynputscrolling - (works better than pyautogui on browsers)
5.mathcalculating - distance between landmarks
